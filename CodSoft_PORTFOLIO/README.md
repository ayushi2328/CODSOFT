# Personal Portfolio Website

## 📌 Project Overview

This is a responsive Personal Portfolio Website developed using HTML and CSS. The website showcases my profile, technical skills, projects, resume, and contact information in a clean and user-friendly layout.

## 🚀 Features

* Responsive design
* About Me section
* Skills section
* Projects showcase
* Resume download option
* Contact information
* Clean and modern user interface

## 🛠️ Technologies Used

* HTML
* CSS3

## 📂 Project Structure

Portfolio/
├── index.html
├── style.css
├── profile.jpg
└── resume.pdf

## 💡 Learning Outcomes

Through this project, I learned:

* Web page structuring using HTML
* Styling and layout design using CSS
* Flexbox implementation
* Responsive design principles
* Portfolio website development

## 📸 Output

The portfolio website displays personal information, skills, projects, resume, and contact details in an organized manner.
