# Landing Page Project

## Overview

This project is a responsive landing page developed using HTML and CSS. The website represents a fictional platform named CodeLaunch that helps students learn coding, build projects, and gain internship opportunities.

## Features

* Responsive Navigation Bar
* Hero Section
* Feature Cards
* About Section
* Contact Section
* Footer

## Technologies Used

* HTML5
* CSS3

## Learning Outcomes

* Page Layout Design
* Flexbox
* Responsive Design
* CSS Styling
* UI Development
