# Calculator Web Application

## 📌 Project Overview

This project is a responsive Calculator Web Application developed using HTML, CSS, and JavaScript. It allows users to perform basic arithmetic operations through an interactive and user-friendly interface.

The project demonstrates the implementation of front-end web development concepts, including user interface design, event handling, DOM manipulation, and JavaScript-based calculations.


## 🚀 Features

* Addition, Subtraction, Multiplication, and Division
* Clear (C) button to reset the display
* Backspace (⌫) functionality
* Decimal number support
* Interactive button hover effects
* Responsive and modern user interface
* Error handling for invalid expressions


## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)


## 📂 Project Structure

Calculator/
│

## ⚙️ How It Works

1. Users enter numbers and operators by clicking calculator buttons.
2. JavaScript updates the display dynamically using DOM manipulation.
3. Mathematical expressions are evaluated when the "=" button is pressed.
4. The result is displayed instantly on the screen.
5. Invalid expressions are handled gracefully and display an error message.
